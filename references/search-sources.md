# China Search Source Map

## Search engines

| Source | Use | Notes |
|---|---|---|
| Baidu | Main mainland web discovery | Use Chinese queries; verify results on destination pages. |
| 360 Search / so.com | Cross-check Baidu results | Useful for enterprise/security-related pages. |
| Sogou WeChat | Public WeChat official account articles | Manual/public use only; do not bypass expired signatures or anti-bot. |
| Bing China | Supplemental desktop/global search | Useful for official docs and GitHub. |

## Legal/risk sources

| Source | Use | Notes |
|---|---|---|
| Tianyancha/Qichacha/Aiqicha | Business registration, shareholders, lawsuits, penalties | Use public/manual or official API only; avoid VIP scraping. |
| Official regulator sites | Notices, penalties, standards | Prefer official. |
| CVE/NVD/GitHub advisories | Vulnerability checks | Use for software/vendor risk. |
| Negative search terms | Complaints, penalties, data leaks | Verify with official or multiple sources. |

## Source reliability rule

1. Official regulator/vendor/API docs > official GitHub/Gitee/org repo > licensed database > reputable media > technical community > marketplace/social/community.
2. Use community content for leads, not final conclusions.
3. Keep original Chinese text and Vietnamese translation together.
